# ✦ 벤티첼로 델피챔바의 타로

> 제천 감성 숙소 [벤티첼로 델피참바](https://naver.me/FMTTbYF0)에서 당신의 이야기를 들려드립니다.
> 78장의 라이더-웨이트-스미스 타로 덱으로 보는 무료 온라인 타로 리딩.
> 예/아니오부터 켈틱 크로스까지 10가지 스프레드 지원.

## 주요 기능

- **전체 78장 타로 덱** — 메이저 아르카나 22 + 마이너 아르카나 56
- **실제 RWS 이미지** — 파멜라 콜먼 스미스(1909) 공개 도메인
- **10가지 스프레드**
  - 빠른 답: 예/아니오 · 오늘의 한 장
  - 쓰리카드 4종: 과거·현재·미래 / 마음·몸·영혼 / 상황·도전·조언 / 나·상대·관계
  - 중급: 파이브카드(십자) · 관계 스프레드 · 말발굽
  - 고급: 켈틱 크로스
- **정/역방향 해석** — 역방향은 5가지 프레임(차단·과잉·내면화·그림자·미해결 과거) 자동 태깅
- **한국어 학습 가이드** — 바보의 여정 · 숫자 의미 · 4원소 수트 · 역방향 프레임
- **반응형 디자인**
  - 데스크탑: 전통 켈틱 크로스 십자 레이아웃
  - 모바일 세로: 3단계 Step Mode (오버뷰 → 한 장씩 → 전체 돌아보기)
  - 모바일 가로: 좌우 분할 레이아웃
- **PWA 메타** — 홈화면 추가 가능

## 기술

- 순수 정적 사이트 (HTML + CSS + Vanilla JS)
- 빌드 프로세스 없음 · 의존성 없음
- 78장 이미지 약 62MB (RWS 공개 도메인)

## 로컬 실행

```bash
# Python 3
python3 -m http.server 8765

# 또는 Node.js
npx serve .
```

브라우저에서 `http://localhost:8765` 접속.

## 배포

Vercel에 정적 사이트로 배포되도록 구성됨 (`vercel.json`). GitHub 연동 후 자동 배포.
